# Is this script necessary ?
Yes, I have facing a problem on my macOS which leaving me just a 10GB of storage, using CC-Cleaner wasn't enough for me. So I downloaded a DaisyDisk to investigate on its free trial. What I have found is a bulk amount of cache files that after mannually removed them giving me like 60GB of storage.

So I was thinking why not create a script to remove them automatically for me and everyone.
